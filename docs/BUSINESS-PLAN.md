## 三、业务流程（完整版）

### 用户旅程

```
【触达】社交媒体/社群/口碑
    ↓
【落地】H5落地页 / 企微社群 / 朋友推荐
    ↓
【评估】AI健康问卷（5分钟）
    ├─ 处方类 → 症状评估+禁忌筛查 → 推荐医师问诊
    ├─ 药食同源 → 体质辨识（九种体质） → 推荐养生方案
    └─ 保健品 → 营养需求评估 → 推荐补充方案
    ↓
【方案】个性化健康方案（AI+医师审核）
    ↓
【下单】微信支付/支付宝
    ↓
【履约】
    ├─ 处方类 → 互联网医院开方 → 药房配药 → 顺丰到家
    ├─ 药食同源 → 工厂直发 → 顺丰到家
    └─ 保健品 → 仓库存储 → 顺丰到家
    ↓
【随访】
    ├─ 企微健康顾问 1对1 跟踪
    ├─ 每周打卡/效果反馈
    └─ AI自动调整方案
    ↓
【复购】Day 21 自动提醒 → 一键续费 → 自动发货
    ↓
【裂变】邀请好友 → 双方优惠 → 社群运营
```
